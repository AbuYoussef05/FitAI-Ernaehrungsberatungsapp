## Die Idee
* Ernährungsberatungsapp 
* User trägt ein
  * Jetziges Gewicht
  * Ziel(Gewicht, Muskelaufbau/Abnahme/Zunahme)
  * Tage pro Woche
  * Allergien, Ernährungsform
  * Zeitraum
  * Workout
  * Trainingspläne

* Programm gibt aus
  * Kalorien
  * Workouts (und verbrannte Kalorien)
  * Schrittzähler

* KI analysiert und gibt aus
  * Workouts
  * Essensplan für eine Woche mit Nährstoffen
  * Einkaufsliste (mit API zu Picknick?)
  * Fortschrittsanalyse
  * bei Abweichungen -> neue Berechnung der Daten und der Ziele
  * analysiert Muster gibt ggf. Warnungen
  * Digitaler Coach (anpassbarer Typ)

* Gamification System?
